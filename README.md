### 功能

**正常使用**

- 有线、无线网络
- 蓝牙(iPod pro 有时没有声音，需重新打开蓝牙；其他蓝牙音箱无问题)
- USB(2.0、3.0)
- 睡眠唤醒
- 声音前后输出
- 独显硬件加速
- iMessage、iCloud
- 接力、粘贴板

**不能使用**

- 前后音频输入、内置音频(可用usb或者蓝牙代替)
- 随航、隔空投送(网卡限制)
